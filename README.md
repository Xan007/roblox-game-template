# Roblox Game Template

A base template for Roblox games with pre-configured systems for data management, networking and audio.

## Overview

It includes a solid foundation with data persistence, real-time synchronization, networking and audio management all set up and ready to use.

## What's Included

### Data Management
- **ProfileStore** - Saves and manages player data
- **Charm + CharmSync** - Keeps data synchronized between client and server in real-time

### Networking
Uses GreenTea to validate arguments.

### Developer console
- **Conch** - Developer console for debugging and development

### Audio
- Audio manager organized by sound categories, each linked to sound groups for easy control

### Framework
- **lumin/framework** - The core architecture that holds everything together

### Data Updates
- Built-in system for migrating and updating player data safely
